# Leftover-Hub
A recipe recommandation app based on Flutter, using TensorFlowLite for ingredient image recognition. Text and voice input available as multi-modal function.

## Introduction
In 21th century, where a lot of food is wasted and constant financial hardship of student end up with trying their best to reduce consumtion on food. Leftover-Hub helps you to find a good recipes based on your "leftovers"
