# Leftover-Hub
A recipe recommandation app based on Flutter, using TensorFlowLite for ingredient image recognition.

## Overview
**Leftover Hub** is a mobile application that helps users discover recipes based on the ingredients they already have.  
The app integrates **image recognition**, **speech-to-text (STT)**, and **OpenAI’s GPT-3 and DALL-E APIs** to provide intelligent, creative recipe suggestions.

## Objective
Allow users to effortlessly submit their ingredients using below.
- **Image recognition**
- **Voice input (Speech-to-Text)**
- **Simple text entry**

By leveraging AI models and multimodal input methods, the app streamlines how users manage ingredients and discover meals.

## Tech Stack
- **Languages:** Python, Dart  
- **Frameworks:** Flutter, TensorFlow, TensorFlow Lite  
- **APIs:** OpenAI GPT-3, DALL-E
